This project is a Python GUI Project . Its called Sysinfo Meter . This is used to tarck system resoures like RAM, CPU Precentage,
Disk Space, Battery Status etc. in real time . 

This application is a cross-platform application . 

This project is developed using Python 3.4 and tkinter framework

Requird Modules for running this project :

1. shutil
2. psutil
3. tkinter
4. pywin32
